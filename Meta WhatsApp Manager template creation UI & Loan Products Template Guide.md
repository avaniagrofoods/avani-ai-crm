# Meta WhatsApp Manager template creation UI & Loan Products Template Guide

Based on the WhatsApp Manager interface shown in your screenshot, here is the step-by-step guide explaining how to fill out each field, followed by ready-to-use templates designed specifically for your loan products to ensure instant Meta approval.

---

## 1️⃣ How to Fill Out Your Template (Step-by-Step)

Here is the explanation for every option visible in the **WhatsApp Manager > Create Template** wizard screen:

### 🗳️ Category Selection (Top Bar)
* **Marketing:** Use this for welcome messages, special interest rate offers, product announcements, and follow-ups. (Meta allows variables here, but no hard-selling or spammy language).
* **Utility (Recommended for document follow-ups):** Use this for transactional messages. (e.g. sending a document collection reminder, payment links, application updates, or details of a loan request). **Utility templates get approved much faster.**
* **Authentication:** Only for security login OTPs. (Not applicable for loan inquiries).

### 📝 Template Type Options (Left Menu)
* **Default (Selected in screenshot):** Use this for standard text messages (with optional headers, images, body text, footers, and quick reply buttons). **This is what you should select for standard follow-ups.**
* **Flows:** Use this if you are attaching an interactive WhatsApp form (e.g. asking the customer to select loan amount/income directly in chat).
* **Order Status / Order Details:** Used for e-commerce purchases (not needed for loans).
* **Calling permissions request:** A formal message asking if you are allowed to call the customer.

## 3️⃣ Steps to Submit Templates in Meta Manager

Once you've filled out the fields matching the steps above, complete these final actions:

1. **Click Next:** Click the blue **Next** button on the bottom right (as highlighted in your screenshot).
2. **Assign Variable Samples:**
   * Meta will show a screen displaying `{{1}}`, `{{2}}`, etc.
   * You **MUST** fill in the text boxes next to them with realistic sample data (e.g. `{{1}}` = `Rajesh Kumar`). **Failing to do this will result in automatic rejection.**
3. **Submit for Review:** Click **Submit**. Meta's AI system will evaluate the template immediately. Approved templates usually turn green within 1-5 minutes!
